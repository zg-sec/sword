# ScanSword 项目介绍

ScanSword 是一款基于 Wails、Go、React 和 Nuclei 的本地化安全测试桌面工作台，面向安全研究、红蓝对抗、渗透测试和日常漏洞验证场景，帮助用户把 POC 管理、扫描执行、流量复核、请求调试、指纹识别和 AI 辅助规范化收口到同一个桌面应用里。

它不是单一的扫描器，而是一套围绕“模板管理 -> 批量验证 -> 命中复核 -> 流量分析 -> 重放验证 -> 指纹沉淀”的完整工作流。

## 核心能力

### POC 管理

- 支持本地 POC 列表管理、分页浏览、关键词搜索和多维筛选。
- 支持 `severity:`、`tag:`、`author:` 等结构化高级搜索。
- 支持本地文件和目录导入，并展示导入进度、统计与失败原因。
- 兼容常见 Nuclei 模板，以及部分旧式规则和社区差异写法。
- 支持模板编辑、详情查看、删除、批量管理与批量扫描。

### 扫描与复核

- 支持批量扫描任务创建、重扫、停止和结果导出。
- 支持配置目标、代理、速率、超时、请求头和抓包参数。
- 支持扫描结果详情查看、请求包/响应包查看和发送到重放。
- 结果导出通过系统保存对话框完成，默认文件名带时间戳，降低覆盖风险。

### 流量日志与调试

- 统一展示主动发包流量和被动流量日志。
- 支持按来源分类、请求方法、状态码和 POC 归属筛选。
- 支持流量详情查看、自动刷新、手动刷新和清空。
- 支持将请求或扫描命中记录发送到重放页面继续验证。

### 指纹识别

- 支持指纹探测和指纹管理两个独立页面。
- 支持多目标主动探测、代理、协议策略、实时进度和结果导出。
- 指纹模型升级为 `matchers[] + matchers_condition`，兼容旧规则格式。
- 支持指纹增删改查、去重、命中统计和绑定 POC。

### AI 辅助测试

- 提供独立的 `AI辅助 测试` 页面。
- 支持 `规范 POC + 生成指纹 / 仅规范 POC / 仅生成指纹` 三种任务范围。
- 支持 AI 对模板元数据做补全、规范化输出和指纹候选生成。
- AI 配置、缓存、SQLite、报告和产物均独立存放，导入前支持核验。

### 代理池与设置

- 支持独立代理池页面，代理池配置、数据库、订阅和日志彼此独立。
- 支持 4 个本地代理端口、订阅导入、健康检查、自动抓取和质量分布查看。
- 支持基础扫描配置、MITM 证书下载/重置、工作目录打开和授权管理。

## 适合的场景

- 长期维护大量 POC，并希望统一导入、编辑、筛选和批量验证。
- 需要对多个目标持续扫描，并保留完整的命中复核和导出链路。
- 希望把流量日志、请求调试、数据重放和指纹识别串成连续工作流。
- 需要一个本地桌面工具完成模板管理、扫描执行、流量分析和结果导出。

## 当前重点特性

- POC 模板兼容导入与批量管理
- 扫描任务重扫、保存并重新下发、结果导出
- 统一流量日志、来源分类、POC 归属展示
- 请求/响应包查看、数据重放与二次验证
- 指纹探测、指纹管理和命中详情联动
- AI 辅助 POC 规范化与指纹生成工作区
- 代理池独立管理与多端口代理服务

## 平台说明

ScanSword 支持 Windows、macOS 和 Linux 桌面构建。

## macOS 运行问题

![Mac 系统运行问题](./images/mac_error.png)

如果 macOS 提示应用无法打开，通常是系统签名与隔离标记导致。可以在终端执行：

```shell
sudo xattr -dr com.apple.quarantine /Applications/scansword.app
```

注意：`/Applications/scansword.app` 需要替换成你本机实际的应用路径。
